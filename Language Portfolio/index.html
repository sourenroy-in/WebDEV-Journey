Language Portfolio


Language portal for Everyone


HTML

HTML (HyperText Markup Language) is the standard markup language used to structure and define the content of web pages,tags and text, links, images, and multimedia

CSS

Cascading Style Sheets (CSS) is a style sheet language most commonly HTML. It controls the visual appearance of web pages, colors, fonts, layout .

JAVASCRIPT

JavaScript is a versatile, high-level programming language used primarily to create interactive and dynamic content on web pages, working alongside HTML and CSS

PHP

PHP (a recursive acronym for PHP: Hypertext Preprocessor) is a widely-used, open-source, server-side scripting language designed primarily for web development.
